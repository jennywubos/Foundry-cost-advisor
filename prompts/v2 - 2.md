## WHAT THIS APP DOES

A Microsoft field sales / customer success tool that:
1. Simulates Azure AD / Microsoft Entra login (exactly like Azure AI Foundry portal)
2. After login, loads that customer's real-time data (quota, usage, cache match rate, throttling, available capacity)
3. Analyzes that data and recommends the right next offer: **Priority Processing** vs **Provisioned (PTU)**
4. Shows a precise 12-month cost comparison: Standard vs Priority Processing vs PTU

There are **2 demo customer accounts** — each is pre-loaded with different usage data that drives a different recommendation.

## HISTORY GENERATOR FUNCTIONS

```javascript
function generateSpikeyHistory() {
  // 60 data points (1 per minute, last hour)
  // Simulate business-hours pattern with two busy periods and throttle spikes
  return Array.from({length: 60}, (_, i) => {
    const isBusy = (i >= 5 && i <= 20) || (i >= 35 && i <= 52);
    const isThrottlePeak = (i === 14) || (i === 48);
    const rpm = isBusy
      ? 200 + Math.round(Math.random() * 40)
      : 20 + Math.round(Math.random() * 15);
    return {
      min: i,
      rpm,
      tpm: rpm * 15200,
      throttledPct: isThrottlePeak ? 0.32 : (isBusy ? 0.08 : 0.001),
      cacheRate: 0.06 + Math.random() * 0.04,
    };
  });
}

function generateSteadyHistory() {
  // Flat steady-state: consistent 24/7
  return Array.from({length: 60}, (_, i) => ({
    min: i,
    rpm: 215 + Math.round((Math.random() - 0.5) * 12),
    tpm: (215 + Math.round((Math.random() - 0.5) * 12)) * 15200,
    throttledPct: 0.015 + Math.random() * 0.01,
    cacheRate: 0.0,
  }));
}
```
